WELCOME TO PICSTAGRAM! with this instagram clone you can do all the things you love with instagram! 
## Deployment 

https://picstagram2.herokuapp.com/

## User Story 
Every user can 
- Create an account 
- Add a post 
- comment and like other users post 
- Editing and deleting their own post
